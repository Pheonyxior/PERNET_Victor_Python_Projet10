# PERNET_Victor_Python_Projet10
## Créez une API sécurisée RESTful en utilisant Django REST

#### Pré-requis:
- [Poetry](https://python-poetry.org/)

#### Installation
- Cloner ou télécharger le projet dans un dossier au choix.
- Ouvrir un terminal à la racine du projet, rentrer `poetry install` pour installer les dépendances et l'environnement virtuel.
- Activer l'environnement virtuel `poetry env activate`.
- Effectuer les migrations `python .\softdesk\manage.py migrate`.
#### Utilisation
- Ouvrir un terminal à la racine du projet, activer l'environnement virtuel `poetry env activate`.
- Démarrer le serveur `python .\softdesk\manage.py runserver`.
- Pour accéder à l'api: http://localhost:8000/api.
- Pour accéder à l'administration de django: http://localhost:8000/admin.
- Pour créer un nouvel administrateur `python .\softdesk\manage.py createsuperuser`
